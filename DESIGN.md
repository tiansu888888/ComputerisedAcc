# 🎨 Design System: ASE20095 Study Portal

This document outlines the design tokens and layout architecture guidelines used to construct the interactive student portals.

## 🌈 Color Tokens (Macaron Glassmorphism)

- **Pastel Pink:** `#FFD1DC` - Accent highlight, matching card selected border.
- **Pastel Lavender:** `#C3B1E1` - Main header gradients, button backgrounds.
- **Pastel Blue:** `#AEC6CF` - Secondary accent background.
- **Pastel Mint:** `#B0EACD` - Correct answer highlights, success tags.
- **Pastel Yellow:** `#FDFD96` - Warm background glow aura.
- **Ink / Typography:** `#111111` - Used for all active body text to guarantee accessibility contrast compliance on light pastel blocks.
- **Muted Ink:** `#333333` - Secondary labels.

## 📦 Container Design System

- **Glass Containers:**
  - Background: `rgba(255, 255, 255, 0.45)`
  - Backdrop Blur: `filter: blur(40px)`
  - Borders: `1px solid rgba(255, 255, 255, 0.6)`
  - Shadow: `0 8px 32px 0 rgba(31, 38, 135, 0.07)`
- **Aura Effects:** Floating background gradients shifting continuously to provide organic visual depth.

## 🔤 Typography pairings

- **Headers & Titles:** `'Orbitron', sans-serif` - Futuristic digital layout look.
- **Body Content:** `'Rajdhani', sans-serif` - Condensed clean sans-serif for optimal readability.
- **Code Snippets:** `'JetBrains Mono', monospace` - Clean programming block presentation.
